Python数据分析之数据导入

CSV格式是最为常见的数据导入格式，下面介绍一下CSV格式数据的导入：

导入panda库

import pandas as pd

导入本地数据

df = pd.read_csv('/Users/Administrator/Documents/data.csv')

导入Web数据

data_url = "https://raw.githubusercontent.com/alstat/Analysis-with-Programming/master/2014/Python/Numerical-Descriptions-of-the-Data/data.csv"
df = pd.read_csv(data_url)

pandas数据分析库中的read_csv函数能够读取本地和web数据。
