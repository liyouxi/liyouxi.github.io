Dogwood添加主题

Posted on 2016-05-12(星期四) 00:00 in edx

1.克隆主题源代码并修改权限

cd /edx/app/edxapp
mkdir themes
cd themes
sudo git clone -b https://github.com/******/edx-theme
mv edx-theme my_theme
cd appsembler-dogwood/static/sass/
mv _default.scss  _my_theme.scss

2. 修改lms.env.json:

sudo vi /edx/app/edxapp/lms.env.json

修改对应值:

"USE_CUSTOM_THEME": true
"THEME_NAME": "******",

3.编译静态文件

sudo -H -u edxapp bash
source /edx/app/edxapp/edxapp_env
cd /edx/app/edxapp/edx-platform
paver update_assets lms --settings aws

4.重启edxapp

sudo /edx/bin/supervisorctl restart edxapp:
